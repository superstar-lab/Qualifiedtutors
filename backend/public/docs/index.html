Introduction

REST API for access to and minipulation of the Qualified Tutors data model.

This documentation aims to provide all the information you need to work with our API.

Base URL

http://localhost:8000

Authenticating requests

This API is authenticated by sending

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

Laravel session based auth is used for this API. To auth/use the API first make a successful call to the login endpoint and include the value returned in the Set-Cookie header in future requests as a header called Cookie. https://laravel.com/docs/8.x/sanctum#spa-authentication

User Management

Endpoints for registering the various types of user.

Register a new Client type user (student/parent)

Registers a new student for use with the site.

Once registered they will still need to login.
Example request:

Login

Logs user into the application. Successful responses will contain a session cookie that must be presented with future requests.

Example request:
const url = new URL(
    "http://localhost:8000/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "laurie41@example.com",
    "password": "voluptatum"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://localhost:8000/login',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'email' => 'laurie41@example.com',
            'password' => 'voluptatum',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
curl --request POST \
    "http://localhost:8000/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"laurie41@example.com\",
    \"password\": \"voluptatum\"
}"
